# tgrgbace.tv

## How to build

1. Create a `.env` with
```
HMAC_KEY=my-secret-hmac-key
HOSTNAME=my-fully-qualified-domain-name
```

2. `npm install`

3. Enjoy your `dist`

4. running npm install will print a .env file to stdout.  If you'd like to reproduce the results, set those environment variables.
